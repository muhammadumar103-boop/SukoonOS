# SukoonOS Features

This file tracks approved SukoonOS feature areas and their implementation status.

## Feature Status Key

| Status | Meaning |
| --- | --- |
| Proposed | Idea exists, not ready to build |
| Approved | Ready for design and implementation |
| Building | Actively being implemented |
| Shipped | Available in the product |
| Deferred | Intentionally postponed |

## Approval Rule

Before implementation, each feature should have:

- A clear user group.
- A defined workflow.
- Required data entities.
- Access control expectations.
- Success criteria.
- Known privacy or compliance concerns.
